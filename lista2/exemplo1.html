para trabalharmos com html devemos seguir certas regras:

  1. toda tag que é aberta deve ser fachada
  2. a tag mais interna deve ser fechada em primeiro lugar
  3. as tags devm ser escritas em caixa baixa

alguns erros são muito comuns na hora de criar a página

  • quando a página não exibe nada:
    1. title não foi fechado
    2. falta de aspas em style
  • quando os erros não são corrigidos:
  • atenção no código
  • salvar um arquivo e testa outro
