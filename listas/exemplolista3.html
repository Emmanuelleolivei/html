protocolo

na internet tanto o hardware como o software pode ser diferentes.

apesar das diferenças para a troca de informações devemos usar o mesmo protocolo.

podemos definir protocolos como regras para a troca de informações.

protocolos básicos

IP internet protocol
protocolo de internete
responsável pelo endereçamento, pelo destino dos blocos de dados
TCP transfer control protocol
protocolo de controle de transferência
responsável pelo envio dos blocos de dados e integridade do conteúdo

temos outros protocolos como

  1. FTP file transfer protocol responsável pelo download de arquivos
  2. HTTP hipertext transfer protocol
    • protocolo de transferência de hipertexto
    • responsável pela navegação através de links
    • simples que fez o sucesso gerando WWW
  3. SMTP simple mail tranfer protocol protocolo para email básico
