links

sao utilizados para fazer a ligação entre duas páginas
nesta página daremos três exemplos de links:

  1. no primeiro link vamos acessar uma página que está na mesma pasta
  2. o segundo link vai fazer a ligação com uma página de outra página
  3. no último vamos acessar uma página que pode estar em qualquer lugar na web

o link é criado co m a tag "a" composta por:
  • a tag a
  • o endereço da página a ser acessada
  • o texto que será exibido na página
  • atenção a única coisa que é exibida do link na página é o texto
